== Description ==
Provides a Pay upon pick up Payment Gateway for Woocommerce.

This Plugin Will Only Work with Woocommerce Plugin.

This plugin is developed for those who want to offer Pickup products from their store after the order is placed. 

It's easy to use just install this plugin and configure it from Woocommerce Payment Gateway Option.
